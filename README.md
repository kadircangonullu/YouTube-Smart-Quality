# YouTube Smart Quality

Automatically reduce YouTube video quality to **144p** when the tab becomes inactive or the browser window loses focus. When you return, the extension restores the previous video quality automatically.

## Features

* 🎥 Automatically switches to **144p** when YouTube is in the background.
* 🔄 Restores the previous video quality when you return to the tab.
* 💾 Saves bandwidth and reduces unnecessary data usage.
* ⚡ Works automatically with no manual interaction required.

## How It Works

1. Start watching a YouTube video.
2. Switch to another browser tab or minimize the browser.
3. The extension automatically changes the video quality to **144p**.
4. Return to the YouTube tab.
5. Your previous video quality is restored automatically.

## Installation

1. Download or clone this repository.
2. Open **Chrome** and navigate to `chrome://extensions/`.
3. Enable **Developer mode**.
4. Click **Load unpacked** and select the extension folder.

## Permissions

The extension only requires access to YouTube pages in order to detect tab focus changes and adjust video quality.
